# GoIt-react-hw-06-phonebook

## Description

This is a revamped phonebook app built using React and refactored to implement state management using Redux Toolkit. The app assists in organizing and managing your contacts efficiently.

## Installation
To run this project locally, follow these steps:

1. Clone the repository to your local machine using Git:

git clone https://github.com/annaistochnikova/goit-react-hw-06-phonebook.git

2. Navigate to the project directory:

cd goit-react-hw-06-phonebook

3. Install the project dependencies using npm:

npm install

4. Start the development server:

npm start

5. Open your web browser and go to http://localhost:3000 to use the phonebook application.
